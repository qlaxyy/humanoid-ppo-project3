## 记录规范

- 每次改算法参数前先提交 Git，再训练。
- Run ID 使用 `runs/` 下的目录名。
- `Actual Steps` 以 `runs/<run_id>/metadata.json` 的 `last_num_timesteps` 为准。
- `Mean Raw Reward` 使用 `evaluate.py` 输出的 `mean_reward`，不要使用训练中的 normalized reward。
- 备注里记录是否断点续训、Colab 是否中断、是否出现依赖或环境异常。
